## add file to existing zip

```
zip -u <existing_zip>.zip <to_be_added_file>
```

```
zip -ur <existing_zip>.zip <to_be_added_dir>
```

## delete file from existing zip

```
zip -d <existing_zip>.zip <to_be_deleted_file>
```
