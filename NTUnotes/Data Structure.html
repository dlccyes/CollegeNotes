Properties of Big-Oh Notation

Assume all functions are positive.

  • Constant factors can be ignored.
    • kT(n)∈Θ(T(n)),∀k>0kT(n) \in \Theta(T(n)), \forall k > 0kT(n)∈Θ(T(n)),∀k>0
  • Higher power grows faster
    • nr∈O(ns),∀r≤sn^r \in O(n^s), \forall r \leq snr∈O(ns),∀r≤s
  • Fastest growing term dominates a sum
    • f(n)∈O(g(n))⇒f(n)+g(n)∈Θ(g(n))f(n) \in O(g(n)) \Rightarrow f(n) + g(n) \in \Theta(g(n))f(n)∈O(g(n))⇒f(n)+g(n)∈Θ(g(n))
    • Ex. 5n2+27n+1005∈O(n2)5n^2+27n+1005 \in O(n^2)5n2+27n+1005∈O(n2)
  • Polynomial's growth is determined by leading term.
    • If f(n)f(n)f(n) is a polynomial of degree ddd, then f(n)∈Θ(nd)f(n) \in \Theta(n^d)f(n)∈Θ(nd).
  • Transitivity
    • If f(n)∈O(g(n))f(n) \in O(g(n))f(n)∈O(g(n)) and g(n)∈O(h(n))g(n) \in O(h(n))g(n)∈O(h(n)), then f(n)∈O(h(n))f(n) \in O(h(n))f(n)∈O(h(n))

Example 2:

def maxSubSum(myArray):
    maxSum = 0
    idx_start = -1
    idx_end = -1
    for i in range(0, len(myArray)):
        for j in range(i, len(myArray)):
            thisSum = 0
            for k in range(i,j+1):
                thisSum = thisSum + myArray[k]
            if thisSum > maxSum:
                maxSum = thisSum
                idx_start = i
                idx_end = j
    return maxSum,idx_start,idx_end

n = len(myArray)
