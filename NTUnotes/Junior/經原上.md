## miscellaneous
- in a competitive industry some firms can ears positive economic profits while some others earn 0 in the long run bc firms have different cost structures
- 可能影響 [[price elasticity]] 大小的因素
	- 替代品的多寡
	- 該商品佔消費支出比重
	- 反應時間長短
	- 物品耐用性
	- 商品定義範圍
- 請以 [[price elasticity]] 討論"穀賤傷農"背後的原因維和？
	- 稻米對消費者是必需品，[[price elasticity]] 較小，因此穀價下跌時，需求上升幅度並不高，導致農夫總收入降低
