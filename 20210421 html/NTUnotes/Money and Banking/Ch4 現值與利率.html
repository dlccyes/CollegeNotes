名詞

  • 期末價值 FV, Final Value
    • 現在的錢在 n 年後值多少
    • FVn=PV×(1+i)nFV_n = PV\times(1+i)^nFVn​=PV×(1+i)n
      • i = 年利率
  • 現值 PV, Present Value
    • n 年後的錢在今天值多少
    • PV = FVn(1+i)n\dfrac{FV_n}{(1+i)^n}(1+i)nFVn​​
  • 內部報酬率 internal rate of return
    • 一項投資相當於年利率多少

信用工具

  • 簡單借貸 simple loan
    • 借出一筆金額、到期日一次還本利
  • 固定給付 fixed-payment loan
    • 借出一筆金額，分期還固定金額
    • 分期攤提制 amortization
  • 息票債券 coupon bond
    • 借出一筆金額，分期還利息，到期日還面額
      • 面額
      • 息票率 coupon rate：每次還的利息 = 面額 x 息票率
    • 永續債券 consol
      • 無到期日，付利息直到永遠
      • PV = Ci\dfrac{C}{i}iC​ （等比）
      • e.g. UK 於拿破崙戰爭時發行
    • 零息票債券 zero-coupon bond
      • 無利息，到期日還面額
      • e.g. 國庫券

到期殖利率 yield to maturity

  • 即內部報酬率
  • 息票債券價格 = 面額時，息票率 = 到期殖利率
    • imagine 把錢存到銀行

其他收益率

  • 目前收益率
    • i=CPi=\dfrac{C}{P}i=PC​
      • C = 利息
      • P = 價格
    • 相當於永續債券息票率
  • 折價基礎收益率 yield on a discount basis
    • idb=F−PF×360di_db=\dfrac{F-P}{F}\times\dfrac{360}{d}id​b=FF−P​×d360​
      • F = 面額
      • P = 價格
      • d = 距到期天數
    • 極短期估價用 ~by~ ~me~

持有期報酬率

  • 未到期即賣出，持有期間之報酬率
  • 資本利得率 rate of capital gains
    • g=Pt+1−PtPtg=\dfrac{P_{t+1}-P_t}{P_t}g=Pt​Pt+1​−Pt​​

實質利率&名目利率

  • 通貨膨脹率 π=Pt+1−PtPt\pi=\frac{P_{t+1}-P_t}{P_t}π=Pt​Pt+1​−Pt​​
  • 實質利率 real interest rate
    • 考慮通貨膨脹
    • 費雪方程式：i≈πe+iri\approx\pi^e+i_ri≈πe+ir​
      1. 1+ir=1+i1+π1+i_r=\dfrac{1+i}{1+\pi}1+ir​=1+π1+i​
      2. πir\pi i_rπir​可省略
      3. i≈π+iri\approx\pi+i_ri≈π+ir​
      4. π為事後得知通膨率\pi為事後得知通膨率π為事後得知通膨率，πe為事前預測通膨率\pi^e為事前預測通膨率πe為事前預測通膨率
  • 名目利率 nominal interest rate
    • 不考慮通貨膨脹
