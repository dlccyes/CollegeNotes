increasing returns to scale: output 變動程度較規模變動程度高 → 邊際成本遞減
constant returns to scale: 相同 → 邊際成本不變
decreasing returns to scale: 較低 → 邊際成本遞增

到最後通常為 decreasing 因為有些東西不能等規模增加如 CEO
