## set up DNS record
- for ip
	- add a record -> `Type` = `A`, `Target` = `<your target ip>`
	- `Name` = whatever, it's your subdomain name<br>e.g. `Name` = `www` -> `www.<domain name>` points to `Target`
		- can leave it blank
